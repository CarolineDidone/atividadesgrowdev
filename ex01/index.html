Introdução ao HTML

O que é HTML?

HTML é a sigla para HyperText Markup Language, que em português significa Linguagem de Marcação de Hipertexto. Ele é a base da construção de páginas na web. Com o HTML, é possível estruturar o conteúdo de um site, como textos, imagens, links, vídeos, botões, entre outros elementos. Ao contrário de linguagens de programação, o HTML não executa ações ou processa dados — sua função é organizar e marcar os elementos que aparecem na tela. Ele diz ao navegador o que é cada parte da página: um título, um parágrafo, uma lista, uma imagem, etc.

Principais Tags de Formatação

A tag <b> é usada para deixar o texto em negrito, mas sem dar ênfase semântica. Ou seja, ela só muda o visual, sem indicar que aquele texto tem importância especial.
Exemplo:
Este é um texto em negrito usando a tag <b>.

A tag <strong> também deixa o texto em negrito, mas com significado semântico, indicando que aquele conteúdo é importante.
Exemplo:
É muito importante fazer o backup dos seus arquivos.

A tag <i> serve para deixar o texto em itálico, normalmente usada para nomes estrangeiros, termos técnicos ou outros conteúdos que devem ser destacados visualmente, sem ênfase semântica.
Exemplo:
Eu adoro tomar um espresso depois do almoço.

A tag <em> também exibe o texto em itálico, mas com ênfas e semântica, mostrando que aquele trecho deve ser lido com entonação ou destaque.
Exemplo:
Eu disse para você fazer isso, não para outra pessoa.

A tag <mark> é usada para destacar uma parte do texto, geralmente com um fundo amarelo, como se fosse marcado com um marca-texto.
Exemplo:
O item mais importante está destacado aqui.

A tag <small> é usada para mostrar um texto em tamanho menor, normalmente para rodapés, observações ou textos secundários.
Exemplo:
Oferta válida até domingo. Consulte os termos e condições.

A tag <del> serve para mostrar que um texto foi removido ou substituído, aparecendo geralmente com um risco no meio.
Exemplo:
O preço original era R$ 150,00, agora é R$ 99,00!

A tag <sub> é usada para subscrito, ou seja, para colocar o texto um pouco abaixo da linha normal — muito comum em fórmulas químicas.
Exemplo:
A fórmula da água é H2O.

A tag é usada para sobrescrito, ou seja, colocar o texto um pouco acima da linha — geralmente usada em potências e notas de rodapé.
Exemplo:
23 é igual a 8.
